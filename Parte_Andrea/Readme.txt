Python file "Project.py" can be opened as executable from its folder at the following conditions:

-to have Python installed on the working machine
-to have "tabulate" Python extension installed (get with 'pip install tabulate')
-to have "qrcode" Python extension installed (get with 'pip install qrcode[pil]')

To function, a text file named "REQUEST.xml" must be present in the same folder. 
File "REQUEST.xml" may be generate executing "DB_Test_Gen".
